5.渐进式web应用开发，替代传统原生app应用，借助service workers技术：
	安全 C 通过 HTTPS 来提供服务来防止网络窥探，保证内容不被篡改。
	渐进式 C 能够让每一位用户使用，无论用户使用什么浏览器，因为它是始终以渐进增强为原则
	响应式 C 适应任何环境：桌面电脑、智能手机、平板电脑，或者其他设备。
	不依赖网络连接 C 通过用 service workers 增强，可以在离线或者低质量网络下工作
	类原生应用 C 有像原生应用般的交互和导航给用户原生应用般的体验，因为它是建立在 app shell model 上的。
	持续更新 C 受益于 service worker 的更新进程，应用能够始终保持更新。
	可发现 C 可识别为“应用程序”，是得益于 W3C manifests 元数据和 service worker 的登记，让搜索引擎能够找到 web 应用。
	可再次访问 C 通过推送通知等特性让用户再次访问变得容易。
	可安装 C 允许用户保留对他们有用的应用在主屏幕上，不需要通过应用商店。
	可链接 C 通过 URL 可以轻松分享应用，不用复杂的安装即可运行。
